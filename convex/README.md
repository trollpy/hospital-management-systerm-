# Convex Backend for HMS

This directory contains all Convex functions, queries, and schema definitions for the Hospital Management System.

## Structure

- `schema/` - Data models and validators
- `functions/` - Mutations (write operations)
- `queries/` - Queries (read operations)

## Collections

- `patients` - Patient records and demographics
- `staff` - Hospital staff and providers
- `appointments` - Scheduled appointments
- `visits` - Patient visits and consultations
- `labOrders` - Laboratory test orders
- `labResults` - Laboratory test results
- `prescriptions` - Medication prescriptions
- `inventory` - Pharmacy and supply inventory
- `billing` - Invoices and payments
